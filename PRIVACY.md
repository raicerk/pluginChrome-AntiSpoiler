# Política de Privacidad – Anti Spam (AntiSpoiler)

Esta extensión NO recopila, almacena ni transmite ningún dato personal del usuario fuera del navegador. Toda la información (palabras bloqueadas) se almacena únicamente en el navegador del usuario mediante la API de almacenamiento local de Chrome.

- No se recolectan datos personales.
- No se envía información a servidores externos.
- No se utiliza para publicidad ni seguimiento.

Si tienes dudas, puedes contactarme antes de instalar la extensión.

Fecha de última actualización: 7 de febrero de 2026
